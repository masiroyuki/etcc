# etcc-EpubtoCbzConverter

## 紹介

Epub形式からCBZ形式にファイルを変換するためのCLIツールです。

Epubファイルの画像を抽出、CBZ形式で保存します。
xhtmlファイルから推測した順番を元に変換を行うためCBZ形式でも画像の並びを維持できます。
